# 변수
- 대입연산자를 통해 생성
```
변수명 = 값(수식, 함수, 라이브러리 등등)
score = 100
kor = 90, eng = 80
total = kor + eng
```
- 타입 : type() 통해 확인
```
정수  실수  문자 참/거짓
int, float, str, bool
```
